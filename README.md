# Portfolio Showcase WordPress Plugin

A custom WordPress plugin that allows users to manage and display portfolio projects using Custom Post Types, meta boxes, and shortcodes.

---

## Features

- Custom Post Type (Portfolio Projects)
- Project Details Meta Box
- Featured Image Support
- Portfolio Grid Layout
- Category Filter Tabs
- Responsive Design
- Live Demo & GitHub Links
- Secure Data Handling
- Easy-to-use Shortcode

---

## Installation

1. Copy the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin from the WordPress Admin.
3. Create portfolio projects from **Portfolio Projects**.
4. Add the shortcode below to any page:

## Shortcode

[portfolio_showcase]

---
